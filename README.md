# GitHub Pages

This folder is ready to publish with GitHub Pages.

## Before publishing

1. Confirm `jiayunli223@gmail.com` is the support email you want shown publicly.
2. Commit and push the `docs` folder to GitHub.

## Publish

1. Open your GitHub repository.
2. Go to `Settings` -> `Pages`.
3. Under `Build and deployment`, set `Source` to `Deploy from a branch`.
4. Choose your main branch and the `/docs` folder.
5. Save.

Your pages will publish at a URL like:

`https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPO_NAME/`
